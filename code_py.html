Currículum del Curso Detallado

Nivel 1: Sistema Base - Fundamentos Esenciales

Feature 1.1: Motor de Ejecución y Sintaxis Primaria

Descripción: Capacidad de escribir y ejecutar los comandos más básicos de Python.

Qué aprenderás:
  • Instalación de Python y VS Code.
  • Tu primer script: `print("Hola Mundo")`.
  • Variables y asignación.
  • Tipos de datos: strings, números, booleanos.
  • Operadores aritméticos básicos.

Entregable esperado: Script `calculadora_basica.py`.

Feature 1.2: Controlador de Flujo Lógico

Descripción: Capacidad de tomar decisiones y repetir acciones en el código.

Qué aprenderás:
  • Operadores de comparación y lógicos.
  • Condicionales: `if`, `elif`, `else`.
  • Bucles: `while`, `for` (con `range()`).
  • Instrucciones `break` y `continue`.

Entregable esperado: Script `adivina_numero.py`.

Feature 1.3: Organizador de Datos Simples

Descripción: Capacidad de almacenar y manipular colecciones de datos.

Qué aprenderás:
  • Listas: creación, acceso, métodos básicos.
  • Tuplas: creación, inmutabilidad.
  • Diccionarios: creación, acceso, métodos básicos.

Entregable esperado: Script `lista_de_tareas.py`.

Feature 1.4: Mecanismo de Reutilización (Funciones)

Descripción: Capacidad de encapsular bloques de código para su reutilización.

Qué aprenderás:
  • Definición de funciones (`def`).
  • Parámetros y argumentos.
  • Valores de retorno (`return`).

Entregable esperado: Refactorizar `lista_de_tareas.py` usando funciones.

Nivel 2: Sistema Mejorado - Python Intermedio

Feature 2.1: Constructor de Entidades (OOP Básica)

Descripción: Modelar conceptos del mundo real como objetos.

Qué aprenderás: Clases, Objetos, `__init__`, atributos, métodos, `self`.

Entregable esperado: Script `gestor_biblioteca_simple.py` con clase `Libro`.

Feature 2.2: Persistencia de Datos y Robustez

Descripción: Guardar datos en archivos y manejar errores.

Qué aprenderás: Archivos de texto, JSON, `try-except`.

Entregable esperado: Modificar gestor para guardar/cargar en JSON.

Feature 2.3: Modularización Avanzada y Entornos Aislados

Descripción: Organizar código en módulos y gestionar dependencias.

Qué aprenderás: Creación e importación de módulos, `venv`, `requirements.txt`.

Entregable esperado: Proyecto con módulo `utilidades.py` y entorno virtual.

Feature 2.4: OOP Avanzada - Herencia y Polimorfismo

Descripción: Crear jerarquías de clases y comportamientos flexibles.

Qué aprenderás: Herencia, sobrescritura de métodos, polimorfismo, `__str__`.

Entregable esperado: Script `sistema_animales.py` con clases `Animal`, `Perro`, `Gato`.

Nivel 3: Sistema Conectado y Probado - Python Aplicado

Feature 3.1: Garantía de Calidad (Testing con Pytest)

Descripción: Escribir pruebas automatizadas para asegurar la fiabilidad del código.

Qué aprenderás: Conceptos de testing, `pytest`, aserciones.

Entregable esperado: Tests unitarios para `utilidades.py` y clases anteriores.

Feature 3.2: Interfaz de Servicio Web (API con FastAPI Básico)

Descripción: Exponer funcionalidades de tu código a través de una API web.

Qué aprenderás: API REST, FastAPI, Uvicorn, Pydantic, JSON.

Entregable esperado: API simple para gestor de biblioteca con endpoints GET y POST.

Feature 3.3: Extractor de Información Web (Scraping Básico)

Descripción: Obtener datos de páginas web de forma automatizada.

Qué aprenderás: `requests`, `BeautifulSoup4`, selección de elementos HTML.

Entregable esperado: Script `scraper_citas.py` para `quotes.toscrape.com`.

Feature 3.4: Automatizador de Tareas del Sistema

Descripción: Escribir scripts que interactúen con el sistema operativo.

Qué aprenderás: Módulos `os` y `shutil`.

Entregable esperado: Script `organizador_descargas.py`.

Nivel 4: Sistema Especializado - Profesionalización

En este nivel, eliges una o varias "suites de features" para especializar tu sistema. El contenido detallado de cada feature se explorará a medida que avanzas en tu especialización seleccionada.

Suite de Features 4.A: Backend Robusto y Escalable

Descripción: Desarrollar aplicaciones backend avanzadas y listas para producción.

Features Clave:
  • 4.A.1: Persistencia Profesional (Bases de Datos con SQLAlchemy)
  • 4.A.2: Seguridad de API (Autenticación y Autorización con OAuth2/JWT)
  • 4.A.3: Desarrollo Asíncrono con FastAPI (`async`/`await`)
  • 4.A.4: Despliegue con Docker y Uvicorn/Gunicorn
  • 4.A.5: Introducción a Microservicios y Caching (Redis)

Entregable Típico: API completa con autenticación, BD relacional, desplegada en Docker.

Suite de Features 4.B: Inteligencia Artificial y Ciencia de Datos

Descripción: Aplicar Python para análisis de datos, machine learning y IA.

Features Clave:
  • 4.B.1: Manipulación y Análisis de Datos (NumPy y Pandas)
  • 4.B.2: Visualización de Datos (Matplotlib y Seaborn)
  • 4.B.3: Machine Learning Aplicado (Scikit-learn: regresión, clasificación)
  • 4.B.4: Introducción al Deep Learning (TensorFlow o PyTorch)
  • 4.B.5: Procesamiento de Lenguaje Natural (NLP) básico

Entregable Típico: Modelo predictivo (ej: clasificación de spam), análisis de sentimiento, dashboard interactivo.

Suite de Features 4.C: Web Scraping y Automatización Avanzada

Descripción: Dominar técnicas avanzadas para extracción de datos web y automatización compleja.

Features Clave:
  • 4.C.1: Scraping Dinámico con Selenium (interacción con JavaScript)
  • 4.C.2: Framework Scrapy para scraping a gran escala
  • 4.C.3: Manejo de proxies, CAPTCHAs (con ética) y user-agents
  • 4.C.4: Automatización de infraestructura con Ansible (Python-based)
  • 4.C.5: Interacción con APIs de servicios en la nube (Boto3 para AWS)

Entregable Típico: Scraper robusto para sitios complejos, scripts de Ansible para configurar servidores.
